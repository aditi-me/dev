Aditi Biswas – Portfolio Website

This repository contains the source code for my personal portfolio website. The website showcases my professional background, technical skills, selected projects, and experience as a WordPress developer, with an emphasis on clean design, performance, and usability.

Overview

The portfolio is a responsive, single-page website built using HTML, CSS, and JavaScript. It is designed to provide a clear and structured overview of my work, skills, and career journey, while maintaining a modern and minimal user experience across devices.

Live Website

The portfolio is deployed using GitHub Pages and is accessible at:

https://aditi-me.github.io/dev/

Features

Responsive and mobile-friendly layout

Light and dark theme toggle

Smooth scrolling navigation

Interactive UI elements and subtle animations

Project showcase section

Skills, experience, and education sections

Contact form with email integration

Links to GitHub, LinkedIn, and Behance profiles

Technologies Used

HTML5

CSS3

JavaScript (Vanilla)

Google Fonts

GitHub Pages

Getting Started

To view the project locally:

Clone the repository:

git clone https://github.com/aditi-me/dev.git

Open the HTML file in a browser:

aditi_biswas_portfolio.html

No additional dependencies or build steps are required.

Customization

The portfolio can be customized by editing:

Content sections directly in the HTML file

Styles within the embedded CSS

Animations and interactions in the JavaScript section

Images, project details, and contact information can be updated directly in the markup.
